## ◈ Open Source Projects

### 🎙️ VoiceAid
> **Fully offline voice assistant for Indian government schemes**

Built for rural citizens with low literacy — runs entirely on-device with zero cloud dependency. Hindi + English, automatic language switching, Metal-accelerated speech on Apple hardware.

`gemma2:2b` · `Whisper.cpp` · `Piper TTS` · `ChromaDB` · `FastAPI` · `Next.js PWA`

### 🧠 RelationOS
> **Local LLM relationship intelligence app with Hinglish support**

Private, offline-first — understands the way Indians actually text. Fine-tuned on real WhatsApp conversation patterns using LoRA on Qwen2 7B.

`Qwen2 7B` · `LoRA / Unsloth` · `Hinglish NLP` · `Local-first`
